# BandiCryptoIOS
IOS용 반디에스앤씨 암호화 라이브러리

### CocoaPods 로 설치하기
프로젝트 폴더에 있는 Podfile에 아래와 같이 BandiCryptoIOS SDK를 추가하세요. Podfile을 저장하고 pod install 커맨드를 실행하세요.
```
pod 'BandiCryptoIOS', '~> 1.0.1'
```

### 최소 요구사항
- iOS 13.0+
